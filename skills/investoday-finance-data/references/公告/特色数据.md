# 公告 / 特色数据

---

## 公告向量搜索

接口路径：`announcement/vector-search`
请求方式：**`POST`**
tool_id：`list_announcement_vector-search`

接口说明：通过自然语言查询文本、股票代码及时间范围（yyyy-MM-dd HH:mm:ss），检索匹配的公告向量数据，返回公告ID、标题、发布日期、相关文本片段及其在原文中的位置信息，适用于基于语义的公告内容深度搜索与投研分析场景。

### 输入参数

**Query 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码 | `002594` |
| `announcementId` | — | integer | 公告ID | `11111` |
| `beginDate` | — | string | 开始日期 | `2025-01-01` |
| `endDate` | — | string | 结束日期 | `2026-01-01` |
| `topK` | — | integer | 返回条数 | `10` |

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `query` | ✅ | string | 查询文本 | `新能源汽车` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `announcementId` | 公告 ID | — |
| `guid` | 公告 GUID | — |
| `title` | 公告标题 | — |
| `publishDate` | 发布日期 | — |
| `chunk` | 文本片段 | — |
| `position` | 片段位置 | — |
| `blockStart` | 文本块起始位置 | — |
| `blockEnd` | 文本块结束位置 | — |
| `embeddingStatus` | 向量处理状态 | — |

### 接口示例

```bash
# Query 可选参数: stockCode, announcementId, beginDate, endDate, topK
investoday-api announcement/vector-search --method POST --body-json '{"query":"新能源汽车"}'
```
