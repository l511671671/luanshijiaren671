# 数据字典

---

## 数据字典搜索

接口路径：`data-dict/search`
请求方式：`GET`
tool_id：`get_data_dict_search`

接口说明：支持按表名、字段名或关键词搜索数据字典，返回匹配的表及字段详细信息（包括中英文名称、数据库归属、模块分类、状态及备注等），并标识当前用户权限及元数据完整性，便于大模型进行数据资产发现、字段映射及元数据管理。

### 输入参数

**Query 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `srchtype` | — | integer | 搜索类型 | `1` |
| `keyword` | — | string | 关键字 | `好股` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `admin` | 是否管理员 | `False` |
| `dictTables` | 搜索表结果(key:表名) | `{GZLTJ_ana_viw_cnt_creat=[{tableId=403, tableName=GZLTJ_ana_viw_cnt_creat}]}` |
| `existFiledEng` | 是否存在字段英文说明 | `False` |
| `existTblEng` | 是否存在表英文描述 | `True` |
| `tables` |  | `None` |

### 接口示例

```bash
# Query 可选参数: srchtype, keyword
investoday-api data-dict/search
```

## 数据字典表详情

接口路径：`data-dict/detail`
请求方式：`GET`
tool_id：`get_data_dict_detail`

接口说明：根据数据字典表标识获取指定表的详细元数据信息，包括表的基本属性（如中英文名称、描述、状态、所属模块等）以及所有字段的详细信息（如字段名、中英文说明、数据类型、主键/唯一/非空约束、默认值、精度等），并返回字段关联的常量枚举列表及其定义，适用于数据治理、数据库文档生成及开发过程中的数据结构查询场景。

### 输入参数

**Query 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `tableId` | ✅ | integer | 数据字典表标识 | `101` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `dictCols` |  | `None` |
| `dictTable` | 数据字典表 | `None` |
| `existEngDesc` | 是否存在字段英文说明 | `True` |

### 接口示例

```bash
investoday-api data-dict/detail tableId=101
```
