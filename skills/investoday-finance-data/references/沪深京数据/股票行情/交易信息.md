## 龙虎榜上榜股票明细

接口路径：`stock/dt-details`
请求方式：**`POST`**
tool_id：`list_stock_dragon_tiger_details`

接口说明：通过股票代码和日期范围（格式为yyyy-MM-dd）查询龙虎榜上榜股票的明细数据，包括上榜日期、股票代码与名称、异动类型代码与名称、偏离值、当日成交量和成交金额，支持分页查询，适用于分析特定股票或时间段内的龙虎榜上榜情况。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 交易日期 | `2019-03-14 00:00:00` |
| `stockCode` | 股票代码 | `002356` |
| `stockName` | 股票名称 | `赫美集团` |
| `abnormalType` | 异动类型 | `日涨幅偏离值达7%的前三只证券` |
| `deviationPct` | 披露值(%) | `12.63` |
| `volume` | 成交量(股) | `2077` |
| `amount` | 成交金额(元） | `314860000.0000` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/dt-details --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
