# 沪深京数据 / 股票行情 / 复权行情

## 股票前复权日行情

接口路径：`stock/adjusted-quotes`
请求方式：**`POST`**
tool_id：`list_stock_adjusted_quotes`

接口说明：通过股票代码查询指定股票的复权行情数据，支持按日期范围筛选，返回包括股票代码、名称、证券类型、市场类型、行情时间、昨收价、开盘价、最高价、最低价、最新价、成交量、成交金额等核心数据，适用于股票历史行情分析和趋势研究。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 交易日期 | `2023-10-26` |
| `stockCode` | 股票代码 | `000001` |
| `stockName` | 股票名称 | `贵州茅台` |
| `prevClosePrice` | 昨收盘 | `12.75` |
| `openPrice` | 开盘价 | `12.75` |
| `highPrice` | 最高价 | `128.75` |
| `lowPrice` | 最低价 | `12.75` |
| `closePrice` | 收盘价 | `15.75` |
| `volume` | 成交量（股） | `12345.67` |
| `amount` | 成交金额（元） | `123456.78` |
| `turnover` | 换手率 | `0.0256` |
| `marketCapFloat` | 流通市值（元） | `1234567890.12` |
| `marketCap` | 总市值（元） | `1234567890.12` |
| `changePct` | 涨跌幅 | `0.025` |
| `peTtm` | 市盈率TTM | `15.8` |
| `pb` | 市净率 | `2.45` |
| `vwap` | 成交量加权平均价 | `123.45` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, beginDate, endDate, pageNum, pageSize, stockCodes
investoday-api stock/adjusted-quotes --method POST --body-json '{"stockCode":"002594","beginDate":"2020-01-01","endDate":"2025-01-01"}'
```

## 股票前复权因子

接口路径：`stock/adjustment-factors`
请求方式：**`POST`**
tool_id：`list_stock_share_adj_factors`

接口说明：通过股票代码和日期范围查询指定股票的历史除权除息事件详情，包括除权除息日、每股派现、送股比例、转增比例、配股比例、配股价以及前复权因子等关键数据，用于股票价格调整和财务分析。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `600256` |
| `stockName` | 股票名称 | `广汇能源` |
| `exDate` | 除权除息日 | `2002-09-12 00:00:00` |
| `cashDividendPerShare` | 每股派现（税前） | `0.5` |
| `stockDividendRatio` | 每股送股比例 | `0.20000000000` |
| `stockTransferRatio` | 每股转增股比例 | `0.5` |
| `rightsRatio` | 每股配股比例 | `0.5` |
| `rightsPrice` | 配股价 | `12.5` |
| `adjFactorBackward` | 单次前复权因子 | `0.8307` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/adjustment-factors --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
