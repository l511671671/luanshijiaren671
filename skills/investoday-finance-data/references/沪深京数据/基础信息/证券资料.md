## 股票基本信息

接口路径：`stock/basic-info`
请求方式：**`POST`**
tool_id：`get_stock_basic_info`

接口说明：通过股票代码查询沪深京A股或B股的基本信息，包括股票名称、全称、所属交易所、上市板块、上市状态、上市日期、退市日期、股票类别、公司ID、总股本、流通股本、A股流通股本、公司办公地址、主营业务以及最新财务报告日期，适用于股票信息查询和基本面分析。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `601881` |
| `exchangeCode` | 市场类型 | `SH` |
| `boardName` | 上市板块 | `主板` |
| `stockName` | 股票名称 | `中国银河` |
| `stockFullName` | 证券全称 | `中国银河证券股份有限公司` |
| `listStatus` | 上市状态 | `1` |
| `listDate` | 上市日期 | `2017-01-23 00:00:00` |
| `delistDate` | 摘牌日期 | `2023-08-15` |
| `stockType` | 股票类别 | `A股` |
| `companyId` | 机构内部ID | `BS123458` |
| `sharesTotal` | 总股本 | `10934402256` |
| `sharesFloat` | 无限售流通股 | `10934402256` |
| `sharesFloatA` | 无限售流通股本 | `7243417623.0000` |
| `officeAddress` | 办公地址 | `北京市丰台区西营街8号院1号楼青海金融大厦` |
| `mainBusiness` | 主营业务 | `(1)许可项目证券业务;公募证券投资基金销售;保险兼业代理业务;证券公司为期货公司提供中间介绍业务;证券投资基金托管。(依法须经批准的项目,经相关部门批准后方可开展经营活动,具体经营项目以相关部门批准文件或许可证件为准)(2)一般项目金银制品销售。(除依法须经批准的项目外,凭营业执照依法自主开展经营活动)(不得从事国家和本市产业政策禁止和限制类项目的经营活动。)(3)有关监管机构批准的其他业务。` |
| `reportDate` | 财务报告日期 | `2025-09-30 00:00:00` |
| `companyGrowthStage` | 公司成长阶段 | `1` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, pageNum, pageSize
investoday-api stock/basic-info --method POST --body-json '{"stockCode":"000001","stockCodes":["000001","600519"],"pageNum":1}'
```

## 证券特别提示

接口路径：`stock/special-notices`
请求方式：**`POST`**
tool_id：`list_stock_special_notices`

接口说明：通过股票代码和日期范围查询沪深京股票的特别提示信息，包括股票名称、上市状态、特别处理状态、变动原因、相关日期等核心公告数据，用于监控股票重大事件和风险提示。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `000001` |
| `stockName` | 股票名称 | `平安银行` |
| `publishDate` | 发布日期 | `2023-08-15` |
| `changeDate` | 变更日期 | `1989-03-20 00:00:00` |
| `endDate` | 截止日期 | `1991-04-02 00:00:00` |
| `listStatus` | 上市状态 | `0` |
| `stStatus` | 特别处理状态 | `0` |
| `changeReason` | 变动原因 | `首次公开发行` |
| `cancelDate` | 取消日期 | `2023-11-15` |
| `cancelReason` | 取消原因 | `客户主动取消` |
| `stCancelAnnounceDate` | 取消特别处理公告日期 | `2023-08-15` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/special-notices --method POST --body-json '{"stockCode":"000001","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
