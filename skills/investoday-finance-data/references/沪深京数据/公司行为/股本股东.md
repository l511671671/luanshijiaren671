## 公司股本变动

接口路径：`stock/capital-stock-changes`
请求方式：**`POST`**
tool_id：`list_stock_capital_changes`

接口说明：通过股票代码和日期范围查询公司股本变动历史，返回变动日期、股票代码、股票名称、总股本、A股股本、B股股本和流通A股股本等核心数据，适用于分析公司资本结构变化和股权变动历史。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `changeDate` | 变更日期 | `2011-12-31 00:00:00` |
| `stockCode` | 股票代码 | `601878` |
| `stockName` | 股票名称 | `浙商证券` |
| `sharesTotal` | 总股本 | `2914701986` |
| `sharesA` | A股 | `3124.75` |
| `sharesB` | B股 | `3.85` |
| `floatSharesA` | 流通A股 | `1234567890.12` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/capital-stock-changes --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
