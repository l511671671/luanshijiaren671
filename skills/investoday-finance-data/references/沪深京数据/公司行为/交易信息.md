## A股大宗交易明细

接口路径：`stock/block-trades`
请求方式：**`POST`**
tool_id：`list_stock_block_trades`

接口说明：通过股票代码和日期范围查询沪深京股票的大宗交易明细数据，返回包括交易日期、股票代码与名称、成交价、成交金额、成交量、买卖方营业部等核心信息，支持分页查询，适用于分析特定股票的大宗交易活动。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 交易日期 | `2004-07-29 00:00:00` |
| `stockCode` | 股票代码 | `600050` |
| `stockName` | 股票名称 | `中国联通` |
| `currencyCode` | 货币编码 | `CNY` |
| `tradePrice` | 成交价（元） | `3.28` |
| `marginShortBalance` | 成交金额（万元） | `4891.79` |
| `tradeVolume` | 成交量（万股） | `1491.4` |
| `buyerBroker` | 买方营业部 | `中国国际金融有限公司上海陆家嘴东路证券营业部` |
| `sellerBroker` | 卖方营业部 | `中国国际金融有限公司北京建国门外大街证券营业部` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/block-trades --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
