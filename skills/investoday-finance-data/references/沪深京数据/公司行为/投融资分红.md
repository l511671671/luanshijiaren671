## 配股发行信息

接口路径：`stock/rights-issues`
请求方式：**`POST`**
tool_id：`get_stock_rights_issues`

接口说明：根据股票代码和公告日期范围查询沪深京股票的配股发行信息，包括配股比例、配股价格、货币种类、股权登记日、除权日、配股上市日及配股数量等关键数据，适用于分析公司融资行为和股权变动。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `announceDate` | 公告日期 | `2023-08-15` |
| `stockCode` | 股票代码 | `600519` |
| `stockName` | 股票名称 | `贵州茅台` |
| `rightsRatio` | 每股配股比例 | `0.5` |
| `rightsPrice` | 配股价(元) | `12.5` |
| `currencyCode` | 货币种类 | `USD` |
| `rightsPriceForeign` | 外币配股价(元） | `12.75` |
| `recordDate` | 股权登记日 | `2024-06-20` |
| `exRightsDate` | 除权日 | `2024-06-20` |
| `rightsListDate` | 配股上市日 | `2024-06-15` |
| `rightsShares` | 配股数量 | `1500.75` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/rights-issues --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
