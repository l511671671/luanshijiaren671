## 业绩报告披露时间表

接口路径：`stock/report-schema`
请求方式：**`POST`**
tool_id：`list_stock_report_schema`

接口说明：通过股票代码和日期范围查询沪深京股票的定期报告（如年报、季报）预披露日程，返回包括股票代码、名称、预计披露日、报告期、实际披露日、首次发布日期和最新预计披露日等关键日程信息，帮助用户跟踪上市公司财报发布计划。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `600622` |
| `stockName` | 股票名称 | `光大嘉宝` |
| `reportDate` | 报告期 | `1990-12-31 00:00:00` |
| `actualPublishDate` | 实际披露日期 | `1992-05-18 00:00:00` |
| `nextReportPublishDate` | 下一报告期披露日期 | `1992-05-18 00:00:00` |
| `prevReportPeriod` | 上一报告期 | `1992-08-18 00:00:00` |
| `samePeriodLastYear` | 去年同期 | `2023-07-15` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/report-schema --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
