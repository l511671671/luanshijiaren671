## 高管持股变动明细

接口路径：`stock/exec-shrhld-chg`
请求方式：**`POST`**
tool_id：`list_stock_executive_shareholding_change`

接口说明：通过股票代码和日期范围查询公司高管及相关人员持股变动明细，返回包括变动日期、股票代码与名称、高管姓名与职务、持股人姓名与关系、变动前后持股数、成交均价及变动原因等核心信息，用于分析内部人交易行为和市场信号。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `changeDate` | 变动日期 | `2006-02-13 00:00:00` |
| `stockCode` | 股票代码 | `600879` |
| `stockName` | 股票名称 | `贵州茅台` |
| `managerName` | 领导层姓名 | `盖洪斌` |
| `positionTitle` | 职务 | `高级管理人员` |
| `holderName` | 股份变动人姓名 | `盖洪斌` |
| `relationToManager` | 与领导层关系 | `本人` |
| `sharesBefore` | 变动前持股数(股) | `150000.5` |
| `sharesChange` | 变动股份数(股) | `19799` |
| `avgPrice` | 成交均价(元) | `12.75` |
| `changeReason` | 变动原因 | `股改对价支付` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/exec-shrhld-chg --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
