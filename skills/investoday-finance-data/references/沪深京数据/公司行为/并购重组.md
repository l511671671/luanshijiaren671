## 公司吸收合并

接口路径：`stock/absorp-mergers`
请求方式：**`POST`**
tool_id：`list_stock_absorption_mergers`

接口说明：通过股票代码和日期范围查询沪深京A股、B股公司的吸收合并事件详情，包括合并方案发布时间、被合并公司信息、换股比例、交易金额、合并前后股本变化、财务数据（如总资产、股东权益、主营业务收入、净利润）以及合并流程中的关键日期（如股东大会决议公告日、合并协议签署日、股份变动公告日等），适用于分析公司并购重组历史、评估合并对价合理性及研究企业资本运作。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `firstPublishDate` | 首次信息发布日期 | `2012-01-20 00:00:00` |
| `stockCode` | 股票代码 | `000001` |
| `stockName` | 股票名称 | `平安银行` |
| `targetCompanyName` | 被合并公司名称 | `平安银行股份有限公司` |
| `mergerPlanPublishDate` | 吸收合并预案公告日 | `2012-01-20 00:00:00` |
| `shareholderMeetingAnnounceDate` | 股东大会决议公告日 | `2012-02-10 00:00:00` |
| `mergerProspectusPublishDate` | 吸收合并公告书发布日 | `2023-08-15` |
| `targetCompanyCode` | 被合并公司代码 | `600123` |
| `establishDate` | 成立日期 | `1995-08-03 00:00:00` |
| `registeredCapital` | 注册资本(元) | `8622824478.0000` |
| `mainBusiness` | 主营业务 | `吸收公众存款；发放短期、中期和长期贷款；办理国内结算等` |
| `industryCode` | 所属行业 | `互联网` |
| `industryName` | 行业名称 | `互联网科技` |
| `mergerBaseDate` | 合并基准日 | `2023-12-31` |
| `mergerAgreementSignDate` | 合并协议签署日 | `2012-01-19 00:00:00` |
| `conversionRatioNumerator` | 折股比例分子 | `1` |
| `conversionRatioDenominator` | 折股比例分母 | `100` |
| `initialTransactionAmount` | 初始交易金额(元) | `150000` |
| `finalTransactionAmount` | 最终交易金额(元) | `12345.67` |
| `mergerCancelledShares` | 合并抵消股份 | `1000000` |
| `issueShares` | 发行股数 | `100000000` |
| `stateSharesIncrease` | 合并增加国家股(万股) | — |
| `legalPersonSharesIncrease` | 合并增加法人股(万股) | — |
| `individualSharesIncrease` | 合并增加个人股(万股) | — |
| `individualSharesHoldingYears` | 合并增加个人股持股年限(年) | — |
| `shareExchangeStartDate` | 换股起始日 | `2024-03-15` |
| `shareExchangeEndDate` | 换股截止日 | `2024-12-31` |
| `shareExchangeRecordDate` | 换股实施股权登记日 | `2012-04-27 00:00:00` |
| `asOfDate` | 截至日期 | `2011-06-30 00:00:00` |
| `totalAssets` | 资产总额(元) | `286393990000` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/absorp-mergers --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
