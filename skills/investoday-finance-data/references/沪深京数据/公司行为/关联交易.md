# 沪深京数据 / 公司行为 / 关联交易

---

## 公司关联交易明细

接口路径：`stock/related-txns`
请求方式：**`POST`**
tool_id：`list_stock_related_transactions`

接口说明：通过股票代码和日期范围查询上市公司关联交易的详细信息，包括交易公告日期、关联方关系、交易金额、定价依据、交易方式、支付方式等核心数据，用于分析公司治理、关联交易合规性及潜在利益输送风险。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `announceDate` | 公告日期 | `1994-04-30 00:00:00` |
| `stockCode` | 股票代码 | `000839` |
| `stockName` | 股票名称 | `国安股份` |
| `transactionIntro` | 交易简介 | `山东国安信息产业有限责任公司(以下简称山东国安)注册资本为800万元,北京国安电气总公司(以下简称国安电气)持有其70%的股份,本公司决定与山东国安的各方股东签署出资转让协议书,受让国安电气所持有的山东国安70%出资中的51%股份,受让价格为408万元。本次交易属关联交易。` |
| `relatedPartyRelation` | 关联关系 | `同一控股公司` |
| `relatedPartyName` | 关联方名称 | `北京国安电气总公司` |
| `transactionAmount` | 交易金额(万元) | `0.0408000000` |
| `currencyCode` | 币种 | `CNY` |
| `pricingBasis` | 定价依据 | `成本加成法` |
| `tradingMethod` | 交易方式 | `购买股权` |
| `paymentMethod` | 支付方式 | `现金` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/related-txns --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
