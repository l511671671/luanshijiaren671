# 沪深京数据 / 公司行为 / 公司调研

## 机构调研活动明细

接口路径：`stock/institutional-research`
请求方式：**`POST`**
tool_id：`list_stock_inst_research`

接口说明：通过指定股票代码和日期范围（格式为yyyy-MM-dd），查询该股票在特定时间段内发生的机构调研活动明细，返回包括调研活动标题、类型（如特定对象调研、分析师会议等）、调研起止日期、发布信息、接待人员、调研机构、活动地点及详细内容等核心数据，便于大模型分析机构投资者对上市公司的关注动态和互动情况。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | — |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `002254` |
| `stockName` | 股票名称 | `泰和新材` |
| `publishDate` | 发布日期 | `2002-01-23 00:00:00` |
| `researchStartDate` | 调研开始日期 | `2002-01-22 00:00:00` |
| `title` | 标题 | `烟台泰和新材料股份有限公司投资者关系活动记录表` |
| `irActivityType` | 投资者关系活动类别 | `1` |
| `receptionPersonTitle` | 接待人员职务 | `4` |
| `companyReceptionPerson` | 上市公司接待人员 | `董事会秘书张三、财务总监李四、证券事务代表王五` |
| `researchLocation` | 调研地点 | `公司会议室` |
| `researchInstitutionType` | 调研机构类型 | `4` |
| `researchInstitutionName` | 调研机构名称 | ` 山东省国资投资公司` |
| `irActivityContent` | 投资者关系活动主要内容介绍 | `<P>投资者：当时为什么到宁夏那面投资芳纶？<BR>　　答：芳纶项目走到宁夏有` |
| `researchBatch` | 调研批次 | `2023` |
| `researchEndDate` | 调研截止日期 | `2024-03-15` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/institutional-research --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
