## 上市公司审计意见

接口路径：`stock/audit_opinion`
请求方式：**`POST`**
tool_id：`list_stock_audit_opinion`

接口说明：支持通过单个股票代码或批量股票代码，结合报告发布时间范围（格式yyyy-MM-dd）、分页参数，查询上市公司的审计意见相关信息，包括报告截止日期、股票代码及名称、审计报告类型、境内外审计机构及服务年限、境内外审计意见（含代码及内容）、注册会计师、审计费用明细及合计、财报净利润与净资产差异金额等核心数据，可用于上市公司财务合规分析、投研中的公司财务质量评估等场景。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2020-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

|--------|------|------|
| `reportPeriodEnd` | 报表截止日期 | — |

| `domesticAuditServiceYears` | 境内审计机构服务时间(年) | — |
| `overseasAuditServiceYears` | 境外审计机构服务时间(年) | — |

## 主要供应商及客户情况

接口路径：`stock/major-sup-cust`
请求方式：**`POST`**
tool_id：`list_stock_major_sup_cust`

接口说明：支持通过单个股票代码、批量股票代码列表，结合可选的报告截止日期范围（格式yyyy-MM-dd）、页码、页大小，查询A股、B股等指定股票的主要供应商与客户相关数据，包括发布日期、报告截止日期、股票代码及名称、前五大供应商采购占比、前五大客户销售占比、前五大供应商采购总金额、前五大客户销售总金额，可用于分析上市公司供应链依赖度、客户集中度等，辅助投研决策。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2020-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `publishDate` | 发布日期 | — |
| `reportPeriodEnd` | 截止日期 | — |
| `stockCode` | 股票代码 | — |
| `stockName` | 股票名称 | — |
| `supplierPurchaseRatio` | 前五大供应商共占年度采购金额的比例(%) | — |
| `customerSalesRatio` | 前五大客户共占销售总额的比例(%) | — |
| `totalSupplierPurchase` | 前五大供应商采购金额合计(万) | — |
| `totalCustomerSales` | 前五大客户销售金额合计(万) | — |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/major-sup-cust --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
