# 沪深京数据 / 财务数据 / 业绩预告

## 业绩快报

接口路径：`stock/earnings-bulletins`
请求方式：**`POST`**
tool_id：`list_stock_earnings_bulletins`

接口说明：通过股票代码和日期范围查询沪深京股票的业绩快报数据，包括营业收入、净利润、每股收益、净资产收益率等核心财务指标的本期值、上年同期值及同比变化，以及报告类型、会计区间、发布日期、截止日期等关键信息，适用于快速获取和分析上市公司定期财务表现。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2020-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `600018` |
| `stockName` | 股票名称 | `上港集团` |
| `publishDate` | 发布日期 | `2007-01-22 00:00:00` |
| `endDate` | 截止日期 | `2006-12-31 00:00:00` |
| `reportType` | 报告类型 | `Q1` |
| `fiscalPeriod` | 会计区间 | `12` |
| `revenue` | 营业收入 | `1261094` |
| `revenueMain` | 主营业务收入 | `1261094.0000` |
| `grossProfit` | 主营业务利润 | `565638` |
| `operatingProfit` | 营业利润 | `-1234567.89` |
| `profitBeforeTax` | 利润总额 | `436282` |
| `netProfitParent` | 归母净利润 | `295000` |
| `netProfitDeducted` | 扣非归母净利润 | `-12345678.9` |
| `epsBasic` | 基本每股收益 | `0.14` |
| `epsWeighted` | 加权平均每股收益 | `3.45` |
| `epsDeducted` | 扣非基本每股收益 | `-0.15` |
| `roe` | 摊薄净资产收益率 | `-0.035` |
| `roeWeighted` | 加权平均净资产收益率 | `0.152` |
| `roeDeducted` | 扣非摊薄净资产收益率 | `-0.035` |
| `roeDeductedWeighted` | 扣非加权平均净资产收益率 | `-0.035` |
| `totalAssets` | 总资产 | `12345678.9` |
| `equityParent` | 归母所有者权益 | `123456789.12` |
| `bps` | 每股净资产 | `1.14` |
| `revenueLyr` | 上年同期营业收入 | `1261094` |
| `revenueMainLyr` | 上年同期主营业务收入 | `1261094.0000` |
| `grossProfitLyr` | 上年同期主营业务利润 | `565638` |
| `operatingProfitLyr` | 上年同期营业利润 | `-1250000.75` |
| `profitBeforeTaxLyr` | 上年同期利润总额 | `436282` |
| `netProfitParentLyr` | 上年同期归母净利润 | `295000` |
| `netProfitDeductedLyr` | 上年同期扣非净利润 | `-12345678.9` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/earnings-bulletins --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
