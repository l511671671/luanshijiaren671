## 资产负债表（时点）

接口路径：`stock/balance-sheets-pit`
请求方式：**`POST`**
tool_id：`list_stk_balan_sheet_pit`

接口说明：通过股票代码查询指定沪深京股票的资产负债表数据，包含母公司期初数、期末数、合并期初数、期末数等核心财务指标，同时提供报告日期、公司性质、科目代码等辅助信息，便于大模型进行财务分析和风险评估。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `publishDate` | 信息发布日期 | `2025-08-30` |
| `stockCode` | 股票代码 | `000001` |
| `stockName` | 股票名称 | `中国平安` |
| `reportPeriodEnd` | 截止日期 | `2025-06-30` |
| `isConsolidated` | 是否合并 | `1` |
| `isAdjusted` | 是否调整 | `1` |
| `sourceName` | 信息来源 | `招股说明书` |
| `sourceCode` | 信息来源编码 | `130106` |
| `announcementType` | 公告类别 | `10` |
| `isComplete` | 完整标志 | `1` |
| `accountingStandard` | 会计准则 | `1` |
| `statementFormat` | 报表格式类型 | `35` |
| `cashEquivalents` | 货币资金/现金及存放中央银行款项 | `123456` |
| `cash` | 其中:货币资金 | `123456` |
| `clientDeposit` | 其中:货币资金:客户资金存款 | `123456` |
| `depositInCentralBank` | 其中:现金及存放中央银行款项 | `123456` |
| `settlementProvi` | 结算备付金 | `123456` |
| `clientProvi` | 其中:客户备付金 | `123456` |
| `lendCapital` | 拆出资金 | `123456` |
| `financing` | 融出资金 | `123456` |
| `depositInInterbank` | 存放同业款项 | `123456` |
| `depositInAssociate` | 存放联行款项 | `123456` |
| `rMetal` | 贵金属 | `123456` |
| `tradingAssets` | 交易性金融资产合计 | `123456` |
| `amongTradingAssets` | 其中:交易性金融资产 | `123456` |
| `amongFinAetAtFValTpl` | 其中:以公允价值计量且其变动计入当期损益的金融资产 | `123456` |
| `derivativeAssets` | 衍生金融资产 | `123456` |
| `refundableDeposit` | 存出保证金 | `123456` |
| `billAccReceivable` | 应收票据及应收账款 | `123456` |
| `billReceivable` | 其中:应收票据 | `123456` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/balance-sheets-pit --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
