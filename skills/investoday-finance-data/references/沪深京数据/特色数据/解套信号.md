## 股票解套信号明细

接口路径：`stock/unwind-signal-details`
请求方式：**`POST`**
tool_id：`list_stk_unwind_signal_de`

接口说明：根据股票代码和信号生成日期范围，查询股票解套信号的详细信息，包括信号生成与失效日期、买入卖出日期与价格、参考买入价、止盈止损价、信号收益率和股票仓位等关键交易数据，适用于分析特定股票在特定时间段内的解套策略和信号表现。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `publishDate` | 发布日期 | `2025-01-17 00:00:00` |
| `stockCode` | 股票代码 | `000001` |
| `stockName` | 股票名称 | `平安银行` |
| `generationDate` | 信号生成日期 | `2025-01-17 00:00:00` |
| `expireDate` | 信号失效日期 | `2025-01-27 00:00:00` |
| `buyDate` | 买入日期(买入信号生成后一天) | `2025-01-20 00:00:00` |
| `sellDate` | 卖出日期(卖出信号生成后一天) | `2025-02-05 00:00:00` |
| `buyPrice` | 买入价(买入信号生成后一天) | `10.9213` |
| `sellPrice` | 卖出价(卖出信号生成后一天) | `10.9213` |
| `returnSignal` | 信号收益率 | `0` |
| `buyPriceReference` | 参考买入价格 | `10.8738` |
| `takeProfitPrice` | 止盈价 | `11.472` |
| `stopLossPrice` | 止损价 | `10.2756` |
| `stockPosition` | 股票仓位 | `0.2` |
| `stopReason` | 止损原因  | `该股票` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/unwind-signal-details --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```

## 股票解套信号统计

接口路径：`stock/unwind-signal-stat`
请求方式：**`POST`**
tool_id：`get_stk_unwind_sig_stat`

接口说明：通过股票代码查询指定股票的解套信号统计信息，包括近三个月、近六个月和近一年的信号次数、胜率、累计收益率以及平均持仓天数等关键指标，用于评估股票解套策略的历史表现和有效性。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `601158` |
| `stockName` | 股票名称 | `重庆水务` |
| `isSignalPeriod` | 是否在信号期内 | `0` |
| `signalCount3m` | 近三月总信号次数 | `6` |
| `signalCount6m` | 近六月总信号次数 | `10` |
| `signalCount1y` | 近一年总信号次数 | `21` |
| `winRate3m` | 近三月信号胜率 | `0.6667` |
| `winRate6m` | 近六月信号胜率 | `0.7` |
| `winRate1y` | 近一年信号胜率 | `0.6667` |
| `returnAvg3m` | 近三月平均收益率 | `-0.0191` |
| `returnAvg6m` | 近六月平均收益率 | `0.002` |
| `returnAvg1y` | 近一年平均收益率 | `0.0509` |
| `holdingDaysAvg3m` | 近三月平均持仓天数 | `7` |
| `holdingDaysAvg6m` | 近六月平均持仓天数 | `7` |
| `holdingDaysAvg1y` | 近一年平均持仓天数 | `6` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, pageNum, pageSize
investoday-api stock/unwind-signal-stat --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"pageNum":1}'
```
