## 股票综合得分

接口路径：`stock/score`
请求方式：`GET`
tool_id：`get_stock_score`

接口说明：通过股票代码获取股票的综合得分数据，包括情绪面得分、财务面得分、赛道得分、技术得分及其行业平均值，同时提供股票所属的申万三级行业代码和名称，适用于股票分析和投资决策。

### 输入参数

**Query 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `000001` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `emotionScore` | 情绪面得分 | — |
| `emotionScoreAvg` | 情绪面得分-行业平均值 | — |
| `financeScore` | 财务面得分 | — |
| `financeScoreAvg` | 财务面得分-行业平均值 | — |
| `idu4Lv1Id` | 申万1级行业代码 | — |
| `idu4Lv1Name` | 申万1级行业名称 | — |
| `idu4Lv2Id` | 申万2级行业代码 | — |
| `idu4Lv2Name` | 申万2级行业名称 | — |
| `idu4Lv3Id` | 申万3级行业代码 | — |
| `idu4Lv3Name` | 申万3级行业名称 | — |
| `industryScore` | 赛道得分 | — |
| `industryScoreAvg` | 赛道得分-行业平均值 | — |
| `score` | 综合评分 | — |
| `scoreAvg` | 综合评分-平均值 | — |
| `skillScore` | 技术得分 | — |
| `skillScoreAvg` | 技术得分-行业平均值 | — |
| `stockCode` | 股票代码 | — |
| `stockName` | 股票名称 | — |

### 接口示例

```bash
investoday-api stock/score stockCode=000001
```

## 股票的年度营业收入(成长能力)

接口路径：`stock/growth-operating-revenues`
请求方式：**`POST`**
tool_id：`list_stock_growth_op_revs`

接口说明：根据指定的股票代码和查询年份数量，获取该股票近几年的营业收入数据，包括各季度（Q1、Q2、Q3、Q4）的营业收入值、上年同期值以及同比增长率百分比，用于分析公司的成长能力和收入趋势。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `000001` |
| `stockName` | 股票名称 | `中国平安` |
| `year` | 年份 | `2025` |
| `latestReportPeriod` | 最新报告期 | `Q1` |
| `valueQ1` | 一季度值(元) | `12345667.11` |
| `valueQ2` | 半年报值(元) | `12345667.11` |
| `valueQ3` | 三季度值(元) | `12345667.11` |
| `valueQ4` | 年报值(元) | `12345667.11` |
| `valueYoyQ1` | 一季度同比值(元) | `12345667.11` |
| `valueYoyQ2` | 半年报同比值(元) | `12345667.11` |
| `valueYoyQ3` | 三季度同比值(元) | `12345667.11` |
| `valueYoyQ4` | 年报同比值(元) | `12345667.11` |
| `yoyQ1Pct` | 一季度同比增长比例(%) | `20` |
| `yoyQ2Pct` | 半年报同比增长比例(%) | `20` |
| `yoyQ3Pct` | 三季度同比增长比例(%) | `20` |
| `yoyQ4Pct` | 年报同比增长比例(%) | `20` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, pageNum, pageSize
investoday-api stock/growth-operating-revenues --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"pageNum":1}'
```
