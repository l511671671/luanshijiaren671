# 沪深京数据 / 特色数据 / 择时信号

---

## 行情技术指标(数据挖掘)

接口路径：`stock/market-technical-indicators`
请求方式：**`POST`**
tool_id：`list_stock_market_ind_dm`

接口说明：根据股票代码和日期范围（格式为yyyy-MM-dd）查询指定沪深京股票在特定时间段内的行情技术指标数据，包括成交量与开盘价相关性、成交量与收盘价排序、开盘价收益率排序、最高价与成交量相关性排序、收盘价收益率、收盘价波动性排序、价量多重排序、最高价波动性排序、最高价与成交量排名相关性、收盘价市值中性化因子、收盘位置因子、迭代收盘相对位置因子以及市值排名迭代收益率因子等核心量化因子，用于股票量化分析和策略研究。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `000001` |
| `stockName` | 股票名称 | `平安` |
| `date` | 交易日期 | `2023-10-26` |
| `volumeOpenPriceCorr` | 成交量-开盘相关性因子 | `1` |
| `volumeFactorClosePriceRank` | 成交量-收盘价纵向排序因子 | `3` |
| `openPriceReturnRank` | 开盘价-收益率截面排序因子 | `0.75` |
| `highPriceVolumeCorrRank` | 最高-成交量相关性横向排序因子 | `0.873` |
| `closePriceReturn` | 收盘价-收益率因子 | `-0.0235` |
| `closePriceVolatilityRank` | 收盘价波动性横向排序因子 | `0.75` |
| `priceVolumeMultiRank` | 价量多重排序因子 | `-0.85` |
| `highPriceVolatilityRank` | 最高价波动性横向排序因子 | `0.85` |
| `highPriceVolumePctCorr` | 最高价-成交排名相关性因子 | `0.78` |
| `closePriceFactorNeutralized` | 收盘价市值中性相关性因子 | `-0.0347` |
| `closePosition` | 收盘位置因子 | `0.85` |
| `closeRelativePositionIter` | 迭代收盘位置因子 | `0.75` |
| `marketCapRankReturnFactorIter` | 市值排名-迭代收益率因子 | `-0.0325` |

### 接口示例

```bash
# Body JSON 可选参数: stockCode, stockCodes, beginDate, endDate, pageNum, pageSize
investoday-api stock/market-technical-indicators --method POST --body-json '{"stockCode":"002594","stockCodes":["000001","600519"],"beginDate":"2020-01-01"}'
```
