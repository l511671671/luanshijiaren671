## 国债收益率

接口路径：`economic/gover-bond-yield`
请求方式：**`POST`**
tool_id：`list_gover_bond_yield`

接口说明：查询中国国债收益率曲线数据，支持按起止日期（格式为yyyy-MM-dd）筛选，返回指定日期范围内各关键期限（3个月、6个月、2年、10年、30年）的国债收益率，用于宏观经济分析、利率走势研判和固定收益投资决策。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 发布时间 | `2025-01-01` |
| `bnd3m` | 三月期债券收益率(%) | `1.5475` |
| `bnd6m` | 六月期债券收益率(%) | `1.5475` |
| `bnd2y` | 两年期债券收益率(%) | `1.5475` |
| `bnd10y` | 十年期债券收益率(%) | `1.5475` |
| `bnd30y` | 三十年期债券收益率(%) | `1.5475` |

### 接口示例

```bash
# Body JSON 可选参数: beginDate, endDate, pageNum, pageSize
investoday-api economic/gover-bond-yield --method POST --body-json '{"beginDate":"2020-01-01","endDate":"2025-01-01","pageNum":1}'
```

## 货币供应量

接口路径：`economic/money-supplies`
请求方式：**`POST`**
tool_id：`list_money_supplies`

接口说明：查询中国货币供应量数据，支持按起止日期（格式为yyyy-MM-dd）筛选，返回指定日期范围内各统计日期的M0、M1、M2的绝对数值、同比及环比增长率，以及M1-M2剪刀差，用于宏观经济分析、货币政策研判和流动性监测。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 截止日期 | `2025-12-30` |
| `m0` | 流通中货币M0(亿元) | `100100` |
| `m1` | 货币M1(亿元) | `100100` |
| `m2` | 货币和准货币M2(亿元) | `100100` |
| `m0YoyPct` | M0同比增长(%) | `20` |
| `m1YoyPct` | M1同比增长(%) | `20` |
| `m2YoyPct` | M2同比增长(%) | `20` |
| `m0QoqPct` | M0环比增长(%) | `20` |
| `m1QoqPct` | M1环比增长(%) | `20` |
| `m2QoqPct` | M2环比增长(%) | `20` |
| `m1M2Spread` | M1-M2剪刀差 | `10` |

### 接口示例

```bash
# Body JSON 可选参数: beginDate, endDate, pageNum, pageSize
investoday-api economic/money-supplies --method POST --body-json '{"beginDate":"2020-01-01","endDate":"2025-01-01","pageNum":1}'
```
