## 闪电诊基

接口路径：`api/prompt/diagnosis-fund`
请求方式：**`POST`**
tool_id：`prompt_diagnosis_fund`

接口说明：闪电诊基API通过输入基金代码和内容丰富度参数，快速生成基金诊断提示词，适用于需要快速获取基金诊断信息的场景。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | ✅ | string | 基金代码 | `159001` |
| `richness` | ✅ | string | 内容丰富度 | `simple` |
| `industryCodes` | ✅ | array | 行业代码 | `['640000', '740000']` |
| `matchType` | ✅ | integer | 匹配类型 | `1` |
| `threshold` | ✅ | number | 阈值 | `40` |
| `isETF` | — | boolean | 是否ETF | `True` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 发布日期 | `2024-03-28 00:00:00` |
| `fundId` | 今日投资内部编码 | `001701JK` |
| `fundCode` | 基金代码 | `001701` |
| `fundName` | 基金名称 | `国联产业升级灵活配置混合` |
| `fundType` | 基金类型代码 | `2` |
| `industryStandard` | 行业体系 | `INDUS4_CL` |
| `industryCode` | 行业代码 | `C` |
| `industryName` | 行业名称 | `制造业` |
| `industryLevel` | 行业级别 | `1` |
| `holdingWeightLatest` | 基于最新报告的持有比重（占基金资产总值） | `43.0181` |
| `holdingWeightQuarterly` | 基于季报的持有比重（占基金资产总值） | `43.0181` |
| `holdingWeightAnnual` | 基于年报的持有比重（占基金资产总值） | `75.8681` |
| `holdingRatioLatest` | 基于最新报告的持有比重（占基金资产净值） | `44.18` |
| `holdingRatioQuarterly` | 基于季报的持有比重（占基金资产净值） | `44.18` |
| `holdingRatioAnnual` | 基于年报的持有比重（占基金资产净值） | `76.84` |
| `holdingRatioLatestPenetrated` | 基于最新报告的持有比重 | `0.125` |

### 接口示例

```bash
# Body JSON 可选参数: isETF
investoday-api api/prompt/diagnosis-fund --method POST --body-json '{"fundCode":"159001","richness":"simple","industryCodes":["640000","740000"],"matchType":1,"threshold":40}'
```

## 闪电诊股

接口路径：`api/prompt/diagnosis-stock`
请求方式：**`POST`**
tool_id：`prompt_diagnosis_stock`

接口说明：通过股票代码和内容丰富度参数，获取闪电诊股的分析结果，适用于快速获取股票诊断信息以辅助投资决策。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |
| `richness` | ✅ | string | 内容丰富度 | `simple` |

### 接口示例

```bash
investoday-api api/prompt/diagnosis-stock --method POST --body-json '{"stockCode":"002594","richness":"simple"}'
```
