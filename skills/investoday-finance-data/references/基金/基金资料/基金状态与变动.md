# 基金 / 基金资料 / 基金状态与变动

---

## 基金发行上市

接口路径：`fund/listings-record`
请求方式：**`POST`**
tool_id：`get_fund_listings_record`

接口说明：通过基金代码或指定首次信息发布日期范围（格式为yyyy-MM-dd HH:mm:ss），查询基金的发行与上市关键信息，包括基金基本信息（代码、名称、内部编码）、发行核心要素（发行对象、面值、发行价格、费用、目标规模、有效认购份额、超额认购倍数、发行状态）、关键日期（首次发布、招股书发布、发行起止、申购赎回开放）以及发行相关统计数据（有效认购户数、管理人参与规模、利息折算份额、认购代码、币种），适用于对基金发行过程、市场表现及合规性进行深度分析和监控。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000001` |
| `fundId` | 今日投资内部编码 | `000001jk` |
| `fundName` | 基金名称 | `华夏成长混合` |
| `firstPublishDate` | 首次信息发布日期 | `2023-08-15` |
| `prospectusPublishDate` | 招募说明书发布日期 | `2023-08-15` |
| `issueTarget` | 发行对象 | `境内合格投资者` |
| `issueStartDate` | 发行起始日 | `2023-05-15` |
| `issueEndDate` | 发行截止日 | `2024-12-31` |
| `parValue` | 基金单位面值 | `1` |
| `issuePrice` | 单位基金发行价格 | `1.05` |
| `issueFee` | 单位基金发行费用 | `12500.75` |
| `issueTotalShare` | 设立募集目标 | `1500000.5` |
| `validSubscriptionShare` | 总有效申购份额 | `1234567.89` |
| `interestConversionShare` | 银行利息折算份额 | `0.0375` |
| `sponsorSubscriptionShare` | 管理人参与规模 | `5000000.75` |
| `subscribeCode` | 申购代码 | `730001` |
| `validSubscriptionAccounts` | 发行有效申购总户数 | `1234567.89` |
| `oversubscriptionMultiple` | 发行超额认购倍数 | `3.75` |
| `subscribeOpenDate` | 申购开放起始日 | `2024-03-01` |
| `redeemOpenDate` | 赎回开放起始日 | `2024-03-15` |
| `currencyCode` | 发行币种代码 | `CNY` |
| `issueStatusCode` | 发行状态代码 | `1` |
| `managementFee` | 管理费率 | `10` |
| `custodyFee` | 托管费率 | `20` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes
investoday-api fund/listings-record --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"]}'
```

## 基金申购赎回状态

接口路径：`fund/subscription-redemption-status`
请求方式：**`POST`**
tool_id：`list_subscription_redemption_status`

接口说明：通过基金代码和日期范围查询基金的申购赎回状态变更历史，返回包括基金代码、名称、变更类型、申购上限、变更日期、适用渠道、对象及备注等关键信息，帮助用户了解基金交易限制的调整情况。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000006` |
| `fundId` | 今日投资内部编码 | `000006JK` |
| `fundName` | 基金名称 | `西部利得量化成长混合A` |
| `changeTypeName` | 变更类型名称 | `开放日常申购` |
| `subscribeLimit` | 申购上限 | `1000000.5` |
| `resumeDate` | 变更日期 | `2019-03-25 00:00:00` |
| `t2.F6H13` | 适用渠道代码 | `ONLINE` |
| `channelName` | 渠道名称 | `微信小程序` |
| `objectType` | 对象代码 | `1001` |
| `objectName` | 对象名称 | `客户订单管理系统` |
| `remark` | 变更说明 | `因业务需求调整，将原定于2024年Q3上线的用户积分商城模块提前至2024年Q2发布，以配合618大促活动。` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/subscription-redemption-status --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"],"beginDate":"2025-01-01"}'
```
