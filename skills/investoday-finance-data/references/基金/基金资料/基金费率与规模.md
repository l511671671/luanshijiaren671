# 基金 / 基金资料 / 基金费率与规模

---

## 基金费率

接口路径：`fund/fee-structures`
请求方式：**`POST`**
tool_id：`list_fund_fee_structures`

接口说明：通过基金代码、费率发布日期范围等条件，查询公募基金（J、K类）的详细费率信息，包括费率类别、币种、适用客户类型、费率范围（最低/最高值）、计算方式、单位、费率划分标准及范围描述、执行状态及日期等核心数据，用于基金费率分析、投资成本评估和产品比较。日期格式为 yyyy-MM-dd HH:mm:ss。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |
