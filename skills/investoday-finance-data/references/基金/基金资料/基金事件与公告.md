## 基金获奖信息

接口路径：`fund/award-records`
请求方式：**`POST`**
tool_id：`get_fund_award_records`

接口说明：通过基金代码查询基金所获奖项信息，返回包括奖项代码、名称、颁奖单位、获奖对象类型、信息发布日期、获奖年度、基金内部代码、基金代码、获奖基金公司代码及名称等核心数据，便于大模型进行基金荣誉评估和奖项分析。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundId` | 今日投资内部编码 | `000001JK` |
| `fundCode` | 基金代码 | `000001` |
| `fundName` | 基金名称 | `华夏成长混合型证券投资基金` |
| `awardCode` | 奖项代码 | `226421828484` |
| `awardName` | 奖项名称 | `十佳基金经理` |
| `awardOrgCode` | 评奖单位代码 | `1001` |
| `awardOrgName` | 评奖单位名称 | `新财富` |
| `awardObjectTypeCode` | 获奖对象类别代码 | `3` |
| `publishDate` | 信息发布日期 | `2002-09-15 00:00:00` |
| `awardYear` | 获奖年度 | `2002-06-30 00:00:00` |
| `companyId` | 获奖基金公司代码 | `FS500003` |
| `companyName` | 获奖基金公司名称 | `华安基金管理有限公司` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes
investoday-api fund/award-records --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"]}'
```
