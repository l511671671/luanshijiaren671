## 基金的现任基金经理及回报

接口路径：`fund/current-manager-returns`
请求方式：**`POST`**
tool_id：`list_fund_current_manager_returns`

接口说明：通过基金代码查询基金的现任基金经理信息，包括基金经理ID、姓名、任职开始时间、任期回报、擅长类型、擅长行情、获奖情况、从业年限、研究领域及从业平均年化表现等核心数据，适用于基金管理和投资分析场景。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000300` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 发布日期 | `2025-10-20` |
| `fundCode` | 基金代码 | `000001` |
| `fundId` | 今日投资内部编码 | `000001jk` |
| `fundName` | 基金名称 | `华夏成长混合` |
| `fundManagerCode` | 基金经理代码 | `001O56LH` |
| `fundManagerName` | 基金经理名称 | `刘睿聪` |
| `employmentBeginDate` | 任职开始时间 | `2024-12-26 00:00:00` |
| `tenureReturn` | 任期回报 | `0.9119` |
| `goodAtType` | 擅长类型 | `1` |
| `goodAtMarketTrend` | 擅长行情 | `1` |
| `careerBeginDate` | 从业开始日期 | `2004-05-21 00:00:00` |
| `yearsOfPractice` |  从业年限（单位：年） | `21.4055` |
| `researchField` | 研究领域 | `4` |
| `avgAnnualizedPerformance` | 从业平均年化 | `0.0269` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes
investoday-api fund/current-manager-returns --method POST --body-json '{"fundCode":"000300","fundCodes":["000001","000006"]}'
```
