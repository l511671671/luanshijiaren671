# 基金 / 基金资料 / 基金经理信息

---

## 基金经理基本信息

接口路径：`fund-manager/basic-info`
请求方式：**`POST`**
tool_id：`get_fund_manager_basic_info`

接口说明：通过基金代码查询指定基金的基金经理基本信息和任职情况，返回包括基金经理的个人ID、姓名、性别、出生日期、最高学历、证书、职位、是否在职、任职起止日期、背景介绍以及关联的基金ID、名称、代码和公告日期等核心数据，适用于基金投研和经理背景分析。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `isIncumbent` | — | integer | 是否在职 | `1` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundId` | 今日投资内部编码 | `000006JK` |
| `fundName` | 基金名称 | `华夏成长混合` |
| `fundCode` | 基金代码 | `000001` |
| `announceDate` | 公告日期 | `2023-08-15` |
| `fundManagerCode` | 基金经理代码 | `P123456789` |
| `fundManagerName` | 基金经理姓名 | `张三` |
| `region` | 地区 | `北京` |
| `gender` | 性别 | `男` |
| `birthDate` | 生日 | `1990-05-15` |
| `educationLevel` | 最高学历 | `博士` |
| `graduationSchool` | 毕业学校 | `深圳大学` |
| `certificate` | 证书 | `ISO9001质量管理体系认证证书` |
| `position` | 职位 | `1` |
| `isIncumbent` | 是否在职 | `1` |
| `startDate` | 任职起始日 | `2023-05-15` |
| `endDate` | 任职截止日 | `2025-12-31` |
| `backgroundDesc` | 背景介绍 | `该客户于2018年成为我行VIP客户，主要经营高端制造业，年营业额约5亿元，信用记录良好，曾成功获得我行两笔总额为8000万元的固定资产贷款并按时偿还。` |
| `partyId` | 机构内部ID | `INS20230915001` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes, isIncumbent
investoday-api fund-manager/basic-info --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"],"isIncumbent":1}'
```
