## 基金业绩比较基准配置

接口路径：`fund/performance-benchmarks`
请求方式：**`POST`**
tool_id：`list_fund_perf_benchmarks`

接口说明：通过基金代码和日期范围查询基金业绩比较基准的配置详情，返回包括基准指数、基准利率、基准权重和年化收益率等关键指标，用于分析基金业绩基准的构成和表现。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 交易代码 | `000001` |
| `fundId` | 今日投资内部编码 | `000001JK` |
| `fundName` | 基金名称 | `华夏成长混合型证券投资基金` |
| `startDate` | 开始日期 | `2023-08-15` |
| `endDate` | 截止日期 | `2024-12-31` |
| `serialNo` | 序号 | `1` |
| `indexCode` | 指数代码 | `000001.SH` |
| `indexName` | 指数名称 | `沪深300指数` |
| `benchmarkRatePct` | 基准利率(%) | `3` |
| `benchmarkRateName` | 基准利率名称 | `贷款市场报价利率(LPR)` |
| `benchmarkWeightPct` | 基准占比(%) | `12.75` |
| `returnAnnPct` | 年收益率(%) | `3.85` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/performance-benchmarks --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"],"beginDate":"2025-01-01"}'
```

## 基金投资标的及比例

接口路径：`fund/investment-targets`
请求方式：**`POST`**
tool_id：`list_fund_invest_targets`

接口说明：通过基金代码及发布日期范围查询基金的投资标的及其配置比例详情，包括标的类别、代码、名称、最大最小投资比例、生效日期、比例基准说明等核心信息，用于分析基金的投资组合结构。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000006` |
| `fundId` | 今日投资内部编码 | `000006jk` |
| `fundName` | 基金名称 | `华夏成长混合` |
| `publishDate` | 信息发布日期 | `2023-08-15` |
| `investTargetType` | 投资标的类别 | `股票` |
| `effectiveDate` | 生效日期 | `2024-03-15` |
| `investTargetCode` | 投资标的代码 | `100001` |
| `investTargetName` | 标的名称 | `沪深300指数期货` |
| `maxRatio` | 最大比例(%) | `0.85` |
| `minRatio` | 最小比例(%) | `0.15` |
| `description` | 投资比例描述 | `持股比例：A公司30%，B公司45%，C公司25%` |
| `ratioBenchmark` | 投资比例基准说明 | `本基金投资于股票资产的比例为基金资产的60%-95%` |
| `remark` | 备注 | `客户要求加急处理，请优先安排发货。` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/investment-targets --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"],"beginDate":"2025-01-01"}'
```
