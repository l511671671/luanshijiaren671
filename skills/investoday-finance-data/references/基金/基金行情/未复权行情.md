## 基金未复权日行情

接口路径：`fund/daily-quotes`
请求方式：**`POST`**
tool_id：`list_fund_daily_quotes`

接口说明：根据基金代码和指定的日期范围（格式为 yyyy-MM-dd HH:mm:ss）查询基金的历史日行情数据，返回包括基金代码、名称、交易日期、开盘价、最高价、最低价、收盘价、成交量、成交金额等核心指标，支持分页查询，适用于分析基金的历史价格走势和交易活跃度。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000006` |
| `fundId` | 今日投资内部编码 | `000006jk` |
| `fundName` | 基金名称 | `华夏成长混合型证券投资基金` |
| `date` | 交易日期 | `2023-10-26` |
| `prevClose` | 前收盘价 | `12.75` |
| `openPrice` | 开盘价 | `12.35` |
| `highPrice` | 最高价 | `128.75` |
| `lowPrice` | 最低价 | `12.75` |
| `closePrice` | 收盘价 | `15.78` |
| `volume` | 成交量 | `12345.67` |
| `amount` | 成交金额 | `123456.78` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/daily-quotes --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"],"beginDate":"2025-01-01"}'
```
