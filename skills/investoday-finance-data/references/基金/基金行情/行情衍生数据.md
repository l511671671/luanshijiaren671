## 基金技术指标

接口路径：`fund/technical-indicators`
请求方式：**`POST`**
tool_id：`list_fund_tech_indicators`

接口说明：通过基金代码和日期范围查询指定基金的技术指标数据，包括压力位和支撑位，用于基金的技术分析和投资决策。入参包括基金代码、开始日期、结束日期、页码和页大小；出参包括基金代码、基金ID、基金名称、交易日期、压力位和支撑位。日期格式为yyyy-MM-dd。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001','000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 交易日期 | `2023-10-26` |
| `fundId` | 今日投资内部编码 | `000006jk` |
| `fundCode` | 基金代码 | `000006` |
| `fundName` | 基金名称 | `华夏成长混合型证券投资基金` |
| `fundAsset` | 基金规模(元） | `1234568.88` |
| `pressure` | 压力位 | `12.75` |
| `support` | 支撑位 | `12.35` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/technical-indicators --method POST --body-json '{"fundCode":"000001","fundCodes":"['\''000001'\'','\''000006'\'']","beginDate":"2025-01-01"}'
```
