## ETF最新实时日行情

接口路径：`fund-quote/realtime`
请求方式：`GET`
tool_id：`get_fund_quote_realtime`

接口说明：通过基金代码获取ETF基金的最新实时行情数据，包括基金代码、名称、市场类型、开盘价、昨日收盘价、当前价格、涨跌幅、最高价、最低价、数据时间和系统时间等核心数据，适用于实时监控ETF基金的市场表现和价格变动。

### 输入参数

**Query 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | ✅ | string | 基金代码 | `159001` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `159616` |
| `fundName` | 基金名称 | `农牧ETF` |
| `marketType` | 市场类型(沪-sh、深-sz、京-bj) | `sz` |
| `openPrice` | 开盘价 | `0.855` |
| `closePriceYDay` | 昨日收盘价 | `0.855` |
| `currentPrice` | 当前价格 | `0.84` |
| `changeRatio` | 涨跌幅 | `-0.0175` |
| `highPrice` | 最高价 | `0.855` |
| `lowPrice` | 最低价 | `0.835` |
| `dataTime` | 数据时间 | `2025-11-04 15:09:00` |
| `sysTime` | 系统时间 | `2025-11-04 15:09:02` |
| `status` | 状态 | `0` |

### 接口示例

```bash
investoday-api fund-quote/realtime fundCode=159001
```
