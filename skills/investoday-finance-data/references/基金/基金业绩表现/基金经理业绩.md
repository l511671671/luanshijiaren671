## 基金经理任职收益

接口路径：`fund-manager/performance`
请求方式：**`POST`**
tool_id：`list_fund_mgr_perf`

接口说明：通过基金代码、基金经理姓名或任职状态，查询公募基金经理的任职收益数据，包括基金经理代码、姓名、任职天数、任职状态、任职收益以及其管理的基金代码、名称和数据截止日期（yyyy-MM-dd HH:mm:ss），用于评估基金经理的历史业绩表现。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `managerName` | — | string | 基金经理姓名 | `张三` |
| `employmentStatus` | — | integer | 任职状态 | `1` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000001.OF` |
| `fundId` | 今日投资内部编码 | `TZ20231015001` |
| `fundName` | 基金名称 | `华夏成长混合` |
| `fundManagerCode` | 基金经理代码 | `JMF001` |
| `fundManagerName` | 基金经理姓名 | `张坤` |
| `returnAnn` | 任职收益 | `12345.67` |
| `endDate` | 数据截止日期 | `2023-12-31` |
| `workingDays` | 任职天数 | `365` |
| `employmentStatus` | 任职状态 | `1` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes, managerName, employmentStatus
investoday-api fund-manager/performance --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"],"managerName":"张三"}'
```

## 基金经理区间回报

接口路径：`fund-manager/interval-returns`
请求方式：**`POST`**
tool_id：`list_fund_mgr_returns`

接口说明：通过基金经理姓名和日期范围查询其在不同时间区间（如本月以来、近一月、近三月、近六月、今年以来、近一年、近三年、近五年）的投资回报率，返回包含基金经理ID、姓名、数据日期、投资类型及各区间回报率的核心数据，适用于评估基金经理的历史业绩表现。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundManagerName` | ✅ | string | 基金经理姓名 | `000001` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundManagerCode` | 人员内部编码 | `004ZSWZL` |
| `fundManagerName` | 姓名 | `许之彦` |
| `date` | 发布日期 | `2013-01-04 00:00:00` |
| `investmentType` | 投资类型 | `0` |
| `returnMtd` | 本月以来回报率 | `0.0009` |
| `return1m` | 近一月回报率 | `0.1751` |
| `return3m` | 近三月回报率 | `0.1037` |
| `return6m` | 近六月回报率 | `0.0357` |
| `returnYtd` | 今年以来回报率 | `0.0009` |
| `return1y` | 近一年回报率 | `0.1253` |
| `return3y` | 近三年回报率 | `-0.2487` |
| `return5y` | 近五年回报率 | `12.75` |

### 接口示例

```bash
# Body JSON 可选参数: beginDate, endDate, pageNum, pageSize
investoday-api fund-manager/interval-returns --method POST --body-json '{"fundManagerName":"000001"}'
```

## 基金经理历史管理基金业绩

接口路径：`fund-manager/hist-performance`
请求方式：**`POST`**
tool_id：`list_fund_mgr_hist_per`

接口说明：支持通过基金经理名称（支持模糊查询）查询其历史管理的基金及任期业绩数据，返回基金代码、基金ID、基金名称、基金经理代码、基金经理名称、是否在职、任职开始日期、任职结束日期（格式均为yyyy-MM-dd）、任期回报等核心信息，可用于评估基金经理的历史任职表现与管理业绩。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundManagerName` | — | string | 基金经理 | `张三` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 发布日期 | `2026-01-01` |
| `fundCode` |  基金代码 | `159355` |
| `fundId` | 今日投资内部编码 | `J000DVGB` |
| `fundName` | 基金名称 | `800红利低波ETF华宝` |
| `fundManagerCode` | 基金经理代码 | `001O57M2` |
| `fundManagerName` | 基金经理姓名 | `胡一江` |
| `isIncumbent` | 是否在任 | `1` |
| `tenureEndDate` | 任职开始日期 | `2025-01-18` |
| `tenureStartDate` | 任职结束日期 | `2025-01-18` |
| `returnTenure` | 任期回报 | `0.1053` |

### 接口示例

```bash
# Body JSON 可选参数: fundManagerName, pageNum, pageSize
investoday-api fund-manager/hist-performance --method POST --body-json '{"fundManagerName":"张三","pageNum":1,"pageSize":10}'
```
