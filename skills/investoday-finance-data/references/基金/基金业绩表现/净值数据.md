## 基金历史净值

接口路径：`fund/nav/history`
请求方式：**`POST`**
tool_id：`list_fund_nav_history`

接口说明：通过基金代码和日期范围查询指定基金的历史净值数据，返回包括基金代码、名称、净值日期、单位净值、累计净值以及发布日期等核心信息，支持分页查询，适用于分析基金的历史表现和净值走势。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000006` |
| `fundId` | 今日投资内部编码 | `000006jk` |
| `fundName` | 基金名称 | `华夏成长混合` |
| `date` | 净值日期 | `2024-03-15` |
| `nav` | 单位净值 | `1.0523` |
| `navAcc` | 累计净值 | `1.356` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/nav/history --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"],"beginDate":"2025-01-01"}'
```

## 货币基金收益情况

接口路径：`fund/currency-yield-history`
请求方式：**`POST`**
tool_id：`list_currency_yield_history`

接口说明：通过基金代码和日期范围（格式为yyyy-MM-dd HH:mm:ss）查询货币基金的收益情况，返回包括基金代码、名称、份额等级、截止日期、万份基金单位收益、七日年化收益率和基金资产净值等核心数据，适用于分析货币基金的历史收益表现和资产规模。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `003171` |
| `fundId` | 今日投资内部编码 | `003171JK` |
| `fundName` | 基金名称 | `信澳慧理财货币A` |
| `date` | 截止日期 | `2019-03-07 00:00:00` |
| `avgDailyProfitPer10k` | 万份基金单位收益 | `0.3822` |
| `avg7dYieldAnn` | 最近七日收益折算的年收益率(%) | `1.187` |
| `nav` | 基金资产净值 | `1234567890.12` |
| `shareClass` | 基金份额等级 | `2` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/currency-yield-history --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"],"beginDate":"2025-01-01"}'
```
