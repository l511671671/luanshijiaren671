## 归因分析

接口路径：`fund/performance-attribution`
请求方式：**`POST`**
tool_id：`get_fund_performance_attribution`

接口说明：通过基金代码查询指定开放式基金（类型为J、K）的综合业绩评估指标，包括基金基本信息（ID、名称、代码）、数据截止日期（yyyy-MM-dd HH:mm:ss格式），以及涵盖成立至今、今年以来、近一周、近一月、近三月、近六月、近一年、近三年、近五年等多个时间维度的年化收益率、累计收益率、Alpha收益、最大回撤、波动率、夏普比率、索提诺比率、卡玛比率、信息比率、特雷诺比率、月度胜率、选股能力、择时能力、跟踪误差等数十项核心风险收益指标，用于全面评估基金的历史表现、风险水平和投资能力。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |

## 业绩比较基准行情

接口路径：`fund/perf-benchmark-quote`
请求方式：**`POST`**
tool_id：`list_perf_benchmark_quote`

接口说明：根据指定的基准指数代码和日期范围（格式为 yyyy-MM-dd HH:mm:ss），查询基金业绩比较基准的历史行情数据，包括基准指数代码、名称、截止日期、收盘价、涨跌额和涨跌幅，用于基金业绩分析和比较。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `benchmarkIndexCode` | — | string | 基准指数代码【与benchmarkIndexCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `benchmarkIndexCodes` | — | array | 基准指数代码列表【与benchmarkIndexCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `benchmarkIndexCode` | 基准指数代码 | `000001` |
| `benchmarkIndexName` | 基准指数名称 | `沪深300指数` |
| `date` | 截止日期 | `2024-12-31` |
| `closePrice` | 收盘价 | `325.78` |
| `chg` | 涨跌额 | `0.25` |
| `changePct` | 涨跌幅(%) | — |

### 接口示例

```bash
# Body JSON 可选参数: benchmarkIndexCode, benchmarkIndexCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/perf-benchmark-quote --method POST --body-json '{"benchmarkIndexCode":"000001","benchmarkIndexCodes":["000001","000006"],"beginDate":"2025-01-01"}'
```

## 基金与指数回报相关系数

接口路径：`fund/index-return-correlations`
请求方式：**`POST`**
tool_id：`list_fund_idx_ret_corr`

接口说明：通过基金代码和日期范围（格式为yyyy-MM-dd HH:mm:ss）查询基金与指数的回报相关系数，返回包括基金代码、名称、日期、内部编码、关联的指数代码以及6个月、1年、3年、5年等多个时间维度的相关系数，用于分析基金与市场指数的历史关联性和风险敞口。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000006` |
| `fundName` | 基金名称 | `华夏成长混合型证券投资基金` |
| `date` | 日期 | `2023-10-26` |
| `fundId` | 今日投资内部编码 | `000006jk` |
| `indexCode` | 指数代码 | `000001` |
| `indexName` | 指数名称 | `上证指数` |
| `corr6m` | 6月相关系数 | `-0.873` |
| `corr1y` | 1年相关系数 | `-0.87` |
| `corr3y` | 3年相关系数 | `-0.327` |
| `corr5y` | 5年相关系数 | `-0.327` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/index-return-correlations --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"],"beginDate":"2025-01-01"}'
```
