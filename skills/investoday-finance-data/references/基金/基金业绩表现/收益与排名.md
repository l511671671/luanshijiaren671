## 基金评价同类平均指标

接口路径：`fund/eval-peer-avg-ind`
请求方式：**`POST`**
tool_id：`get_fund_peer_avg_metric`

接口说明：通过基金代码查询指定基金在多个时间维度（如成立至今、今年以来、最近1周、1月、3月、6月、1年、3年、5年）的全面评价指标，包括该基金的累计收益率、阿尔法收益、年化波动率、最大回撤、下行波动率、夏普比率、索提诺比率、信息比率、特雷诺比率、跟踪误差、贝塔系数、VaR等核心风险收益指标，同时提供各指标在同类基金中的平均值和排名，便于大模型进行基金的横向对比和深度绩效评估。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000300` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |

## 基金回报率

接口路径：`fund/return-rate`
请求方式：**`POST`**
tool_id：`list_fund_return_rate`

接口说明：通过基金代码和日期范围查询指定基金的历史回报率数据，返回包括日回报率、周回报率、月回报率、季度回报率、半年回报率、年回报率以及今年以来回报率等多维度收益指标，用于分析基金在不同时间区间的业绩表现。

### 输入参数

**Query 参数**

_无参数_

**Body JSON 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000006` |
| `fundId` | 今日投资内部编码 | `000006jk` |
| `fundName` | 基金名称 | `华夏成长混合` |
| `returnDaily` | 日回报率 | `-0.0235` |
| `returnYtd` | 今年以来回报率 | `-12.34` |
| `return1w` | 最近1周回报率 | `-0.0345` |
| `return1m` | 最近1月回报率 | `-0.0325` |
| `return3m` | 最近3月回报率 | `-0.0235` |
| `return6m` | 最近6月回报率 | `-0.0345` |
| `return1y` | 最近1年回报率 | `-0.045` |
| `return3y` | 最近3年回报率 | `0.125` |

### 接口示例

```bash
# Body JSON 可选参数: fundCode, fundCodes
investoday-api fund/return-rate --method POST --body-json '{"fundCode":"000001","fundCodes":["000001","000006"]}'
```
