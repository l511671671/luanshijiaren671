## 指数区间涨幅

接口路径：`index/range-gains`
请求方式：`GET`
tool_id：`get_index_range_gains`

接口说明：通过指数代码查询指定指数的多时间区间涨跌幅数据，返回包括指数代码、名称以及近一日、近一周、近一月、近三月、近半年、近一年和今年以来等多个维度的涨跌幅百分比，用于分析指数的短期、中期和长期表现趋势。

### 输入参数

**Query 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `indexCode` | ✅ | string | 指数代码 | `000300` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 指数代码 | `000001.SZ` |
| `stockName` | 指数名称 | `贵州茅台` |
| `return1dPct` | 近一日涨跌幅(%) | `-3.75` |
| `return1wPct` | 近一周涨跌幅(%) | `-3.75` |
| `return1mPct` | 近一月涨跌幅(%) | `-3.75` |
| `return3mPct` | 近三月涨跌幅(%) | `-15.73` |
| `return6mPct` | 近半年涨跌幅(%) | `-15.73` |
| `return1yPct` | 近一年涨跌幅(%) | `-12.75` |
| `returnYtdPct` | 今年以来涨跌幅(%) | `-12.75` |

### 接口示例

```bash
investoday-api index/range-gains indexCode=000300
```

接口路径：`index/price-vol-ind`
请求方式：**`POST`**
tool_id：`list_index_price_volume_indicators`

接口说明：支持通过单个指数代码或批量指数代码，结合指定时间范围（开始日期、结束日期格式均为yyyy-MM-dd），分页获取指数的量价与压力支撑类技术指标数据，包括指数代码、指数名称、交易日期，以及布林带、多空布林线、ENV、MIKE、MFI、OBV、PVT、WVAD、AR、BR、PSY、CR等各类技术指标数值，可用于指数的技术面分析、压力支撑位判断等投研场景。

_无参数_

**Body JSON 参数**

|--------|:----:|------|------|------|
| `indexCode` | — | string | 指数代码【与indexCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |

| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |
