## 港股股本变动

接口路径：`hk-stock/cap_changes`
请求方式：`GET`
tool_id：`list_hk_stock_capital_changes`

接口说明：通过股票代码、起始日期和结束日期查询港股公司的股本变动历史记录，返回包括股票代码、名称、变动日期、各类别股份数量（如优先股、普通股、H股、大陆股等）及总股本等详细信息，适用于分析公司资本结构变化和股权历史。

### 输入参数

**Query 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | — |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `pubDt` | 发布日期 | `2023-11-15` |
| `f2000` | 股票代码 | `00001` |
| `f2004` | 中文简称 | `长和` |
| `f1000` | 公司代码 | `LA00001` |
| `f1100` | 变动日期 | `2024-06-15` |
| `f1101` | 优先股股数 | `5000000` |
| `f1102` | 普通股股数 | `1000000` |
| `f1103` | H股股数 | `1000000` |
| `f1104` | 其他股数（大陆股数） | `10000` |
| `f1105` | 特殊股数 | `10000` |
| `f1106` | 香港股数部分 | `1500.5` |
| `f1107` | 大陆股数部分 | `1000000` |
| `f1108` | 总股数 | `1000000000` |

### 接口示例

```bash
# Query 可选参数: beginDate, endDate, pageNum, pageSize
investoday-api hk-stock/cap_changes stockCode=00001
```
