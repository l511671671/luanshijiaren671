## 港股前复权日行情

接口路径：`hk-stock/adjusted-quotes`
请求方式：`GET`
tool_id：`list_hk_stock_adjusted_quotes`

接口说明：通过股票代码、行情时间或日期范围查询港股复权行情数据，返回包括股票代码、证券类型、交易市场代码、行情日期、前收盘价、开盘价、最高价、最低价、收盘价、成交量、成交金额等核心行情指标，适用于港股市场分析和历史行情回溯。

### 输入参数

**Query 参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

### 输出参数

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockName` | 股票名称 | `谊砾控股` |
| `stockCode` | 股票代码 | `00076` |
| `date` | 交易日期 | `1984-10-04 00:00:00` |
| `prevClosePrice` | 前收盘价（元） | `533599` |
| `openPrice` | 开盘价（元） | `533599` |
| `highPrice` | 最高价（元） | `563253` |
| `lowPrice` | 最低价（元） | `533599` |
| `closePrice` | 收盘价（元） | `543513` |
| `volume` | 成交量（股） | `69` |

### 接口示例

```bash
# Query 可选参数: beginDate, endDate, pageNum, pageSize
investoday-api hk-stock/adjusted-quotes stockCode=00001
```
